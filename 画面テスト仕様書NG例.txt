# 🟡 ワンランク上の書き方（レビュー強者）

### ✨ フォーマット

```
① 画面状態
② 操作結果
③ 表示内容（文言／位置／色）
```

### ✨ 例

```
期待値：
画面遷移せず、
メニュー名入力欄の直下に赤文字で
「メニュー名を入力してください」と表示されること
```

# ✅ NG → OK 早見表（Excel置き換え用）

| NG表現    | 修正キーワード      |
| ------- | ------------ |
| 表示されること | 表示位置＋文言＋色    |
| 正しく     | 画面名／URL／初期状態 |
| 登録される   | 一覧反映＋再表示確認   |
| エラーになる  | 遷移有無＋表示内容    |
| 問題なく    | 何が問題でないか明示   |
